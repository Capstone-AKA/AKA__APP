# AKA__APP
2025 부경대학교 캡스톤디자인 **스마트 쇼핑 시스템 AKA**

이 프로젝트는 오프라인 쇼핑을 스마트하게 지원하기 위해 개발된 앱입니다.

- 스마트 쇼핑 시스템 앱 개발
- 프론트엔드(React Native), 백엔드(Spring Boot), 임베디드 시스템 연동
